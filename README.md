# Shinobu

Web estática para Shinobu, artesanía en madera.

## Publicación en GitHub Pages

1. Sube `index.html`, `catalog/`, `showroom/`, `.nojekyll`, `.gitignore`, `robots.txt` y este `README.md` a la rama `main`.
2. En GitHub, entra en `Settings > Pages`.
3. Selecciona `Deploy from a branch`.
4. Elige la rama `main` y la carpeta `/root`.
5. Guarda los cambios y espera a que GitHub genere la URL pública.

Cuando tengas el dominio, añádelo en `Settings > Pages > Custom domain` y configura los DNS que indique GitHub.
